오픈데이터 코리아란?

오픈데이터 코리아는 대규모의 데이터를 처리하는 과학적 이론(머신러닝, 데이터마이닝, 통계/확률, 가시화 등) 및 공학적인 도구들에 대한 정보를 공유하는 사이트 입니다. 빅데이터 관련 오픈소스는 물론 오픈데이터 API등에 대한 정보를 제공합니다.

빅데이터(Big Data)

오픈소스 도구 카테고리

  • Distributed File System (DFS)
  • MapReduce Framework (MR)
  • Database Storage (NoSQL)
  • Batch Processing (BP)
  • Streaming Processing (SP)
  • Machine Learning (ML)
  • Log Processing (LP)
  • Messasing (MQ)
  • Distributed Coordination (DC)
  • Workflow (WF)
  • Analytic Processing(AL)
  • Scheduling (SH)
  • Cluster Management (CM)
  • Data Serialization (DS)
  • Extract, Transform and Loading (ETL)

오픈소스 도구

  • Hadoop (NoSQL, MR, Apache, inspired by Google's MapReduce): A framework that allows for the distributed processing of large data sets across clusters of computers using simple programming models. YARN (Nextgen Hadoop 2.0)
  • Hive (AL, Apache): Data Warehouse System
  • Pig (AL, Apache): A platform for analyzing large data sets that consists of a high-level language for expressing data analysis programs, coupled with infrastructure for evaluating these programs.
  • Sqoop (ETL, Apache): A tool designed for efficiently transferring bulk data between Apache Hadoop and structured datastores such as relational databases.
  • Flume (SP, Apache): A distributed, reliable, and available service for efficiently collecting, aggregating, and moving large amounts of log data
  • Fluentd (LP, Apache): A tool to collect events and logs.
  • Kafka (MQ, Apache) is publish-subscribe messaging rethought as a distributed commit log
  • Storm (SP, Apache): Distributed and Fault-tolerant real-time Computation
  • Drill (AL, Apache): A distributed system for interactive analysis of large-scale datasets (inspired by Google's Dremel)
  • Spark (MR, Apache): An open source cluster computing system that aims to make data analytics fast — both fast to run and fast to write.

워크플로우

  • Cascading (WF): An application framework for Java developers to simply develop robust Data Analytics and Data Management applications on Apache Hadoop.
  • Oozie (WF, SH, Apache): A workflow scheduler system to manage Apache Hadoop jobs.

분산 데이터베이스

  • Hbase (NoSQL, Apache, inspired by Google's Bigtable)
  • Cassandra (NoSQL, Apache/Datastax)
  • Riak (NoSQL, Basho)
  • Hypertable: An open source database system inspired by Google's BigTable.

관리도구

  • Hue (UI)
  • ZooKeeper (DC): An effort to develop and maintain an open-source server which enables highly reliable distributed coordination
  • StarCluster (CM, MIT)
  • Mesos (CM, Apache): A cluster manager that provides efficient resource isolation and sharing across distributed applications, or frameworks. It can run Hadoop, MPI, Hypertable, Spark, and other applications on a dynamically shared pool of nodes.
