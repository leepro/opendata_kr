오픈데이터 코리아란?

오픈데이터 코리아는 대규모의 데이터를 처리하는 과학적 이론(머신러닝, 데이터마이닝, 통계/확률, 가시화 등) 및 공학적인 도구들에 대한 정보를 공유하는 사이트 입니다. 빅데이터 관련 오픈소스는 물론 오픈데이터 API등에 대한 정보를 제공합니다.

빅데이터(Big Data)

오픈소스 도구 카테고리

  • Distributed File System (DFS)
  • MapReduce Framework (MR)
  • Database Storage (NoSQL)
  • Batch Processing (BP)
  • Streaming Processing (SP)

  • Log Processing (LP)
  • Messasing (MQ)
  • Distributed Coordination (DC)
  • Workflow (WF)
  • Analytic Processing(AL)
  • Scheduling (SH)
  • Cluster Management (CM)
  • Data Serialization (DS)
  • Extract, Transform and Loading (ETL)

오픈소스 도구

  • Apache Hadoop (NoSQL, MR, Google's MapReduce clone): A framework that allows for the distributed processing of large data sets across clusters of computers using simple programming models.
  • Apache Sqoop (ETL): A tool designed for efficiently transferring bulk data between Apache Hadoop and structured datastores such as relational databases.
  • Apache Flume (SP)
  • Fluentd (LP): A tool to collect events and logs.
  • Apache Pig (AL): A platform for analyzing large data sets that consists of a high-level language for expressing data analysis programs, coupled with infrastructure for evaluating these programs.
  • Apache Hive (AL): Data Warehouse System
  • Apache Hbase (NoSQL, Google's Bigtable clone)
  • Apache Kafka (MQ) is publish-subscribe messaging rethought as a distributed commit log
  • Storm (SP) : Distributed and Fault-tolerant real-time Computation
  • Apache ZooKeeper (DC)
  • Cascading (WF)

관리도구

  • StarCluster (CM)

머신러닝 / 데이터마이닝

  • Collaborative Filtering
  • User and Item based recommenders
  • K-Means, Fuzzy K-Means clustering
  • Mean Shift clustering
  • Dirichlet process clustering
  • Latent Dirichlet Allocation
  • Singular value decomposition
  • Parallel Frequent Pattern mining
  • Complementary Naive Bayes classifier
  • Random forest decision tree based classifier
  • High performance java collections (previously colt collections)

시계열 분석 (Time-series Analysis)
